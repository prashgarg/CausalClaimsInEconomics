# Demo Input

Place user input files here for Stage 1-4 extraction runs.

Supported patterns:

- `*.txt` (recommended, first 30 pages extracted to plain text)
- `*.pdf` (convert first with `scripts/convert_pdf_to_text.py`)

Examples:

- `demo_input/paper1.txt`
- `demo_input/paper2.txt`
- `demo_input/paper3.pdf`
